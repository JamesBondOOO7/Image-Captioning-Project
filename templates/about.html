About the Project

This is a quite a complex deep learning model which has the power to create captions, when it is fed with an image.

What it does is, it takes an image as an input and tries to predict an appropriate caption for that image !!

The dataset used in the training of the model is Flickr8k Dataset.

The model has already been trained on gpu. Flask is used for the backend.
